# ALU Back-End - API Project

## Description
This project focuses on manipulating data gathered from third-party REST APIs. It covers accessing system and user information, filtering structured records, and exporting data sets cleanly into alternative management configurations including CSV structures and JSON files.

## Tasks Overview

* **0. Gather data from an API**: Script that returns information about an employee's TODO list progress using a REST API.
* **1. Export to CSV**: Python script extending the tracking system to record all owned tasks into a structured CSV file.
* **2. Export to JSON**: Script configured to port specific employee task objects into standard JSON arrays.
* **3. Dictionary of list of dictionaries**: Consolidates all user tasks across every employee account ID into a single unified JSON output payload mapping file.
